Limitar-se aos modelos convencionais de busca de emprego significa fechar a sí próprio ao mundo. Quando falamos em nos adaptar, falamos em mudar como realizavamos uma determinada tarefa a x anos atrás, e realizar essa mesma tarefa de uma forma mais eficiente.

A busca de emprego a cada dia se torna mais personalizada, onde os pequenos detalhes podem tornar-se mais qualificados ou não para uma determinada vaga. Portanto, apenas enviar currículos e esperar que o telefone toque é algo que ficará no esquecimento em um futuro breve.

Pensando nisso esta página tem como objetivo fazer mais com os recursos que possuo, tornando essa página como um cartão de visitas em constante atualização, tanto para DEVS como para os recrutadores.

Em nosso currículo nos preocupamos em mostrar nossas qualificações e a nossa história como profissionais. Hoje com tantas ferramentas disponíveis nós podemos provar nossos próprios conhecimentos antes mesmo de uma dinâmica de grupo ou teste determinado pela empresa. O GitHub hoje é uma das maneiras mais simples de demonstrarmos nossos conhecimentos para os recrutadores, caso você seja um desenvolvedor, precisa ter um.
